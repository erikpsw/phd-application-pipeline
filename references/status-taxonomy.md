# 套磁状态分类 / Outreach Status Taxonomy

| 状态 | 含义 | 下一步动作 |
| --- | --- | --- |
| 积极回复 | 导师表达可能招生、邀请交流或给出明确积极信号 | 按导师建议准备材料并确定再次联系时间 |
| 流程型回复 | 要求先正式申请，录取前不单独评估 | 按官方流程申请，并在合适节点再次联系 |
| 等待回复 | 自动回复、休假或要求等待 | 等待真实回复，必要时在 2–3 周后跟进 |
| 低优先级 | 资金、匹配度或项目优先级不确定 | 仅在条件改善或缺少更优选择时考虑 |
| 暂不考虑 | 无名额、无资金或本周期不招生 | 礼貌结束沟通，停止当前周期跟进 |
| 已发送 | 首封邮件已发送，尚未收到回复 | 等待 2–3 周后决定是否跟进 |
| 草稿已创建 | Gmail 草稿已创建但未发送 | 人工核对收件人、附件与研究切入点 |

规则：

- 自动回复不能归为积极回复。
- 积极回复不等同于正式录取或资金承诺。
- 回复草稿应短于首封邮件，并遵循导师给出的流程。
